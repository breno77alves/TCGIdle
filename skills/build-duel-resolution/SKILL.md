# Skill: build-duel-resolution

## Purpose
Design or implement a readable duel resolution system for sequential 1v1 battles inside a 6v6 structure.

## Output Format
1. combat objective
2. formula or rules
3. pseudocode
4. implementation notes
5. sample outcomes
6. tuning levers

## Checklist
- readable at a glance
- logs explain what happened
- randomness is controlled
- element advantage is understandable
- victory condition is explicit
- good fit for auto-battle pacing

## Rules
- avoid overcomplicated formulas
- expose tuning parameters clearly
- make debugging easy
