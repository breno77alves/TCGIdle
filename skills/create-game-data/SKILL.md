# Skill: create-game-data

## Purpose
Generate organized, original, easy-to-edit data structures for cards, NPCs, locations, and related game content.

## Use For
- creatures
- attacks
- tribes
- locations
- NPCs
- masters
- Call Cards
- itemized content packs

## Output Format
- a short design note
- structured entries ready for JS objects/arrays
- optional flavor text
- optional visual tags for art prompts

## Rules
- keep naming original
- keep data easy to hand-edit
- keep future expansion in mind without overengineering schema
- ensure content supports gameplay readability
