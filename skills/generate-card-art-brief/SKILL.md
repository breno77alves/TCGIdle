# Skill: generate-card-art-brief

## Purpose
Create high-utility art briefs and prompts for AI-generated creature portraits that remain readable on card fronts.

## Output Format
1. card name
2. subject summary
3. shape/silhouette goal
4. color direction
5. main prompt
6. optional negative prompt
7. thumbnail readability notes

## Prompt Standard
Use prompts that favor:
- one clear subject
- collectible-card appeal
- bright readable forms
- restrained background detail
- family-friendly fantasy/sci-fi presentation
- small-size readability

## Rules
- no text inside the image
- no frames baked into portrait art
- no logos or watermarks
- avoid clutter and tiny details that disappear at card size
