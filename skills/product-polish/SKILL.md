# Skill: product-polish

## Purpose
Raise the perceived quality of a feature or screen so it feels market-ready and intentional.

## Focus Areas
- hierarchy
- spacing
- copy clarity
- surface layering
- button states
- panel composition
- empty states
- result messaging
- card presentation

## Output Format
1. current weakness
2. improvements
3. exact visual/UX changes
4. implementation notes
5. quality checklist

## Rules
- prefer concrete improvements over vague style talk
- prioritize clarity and confidence
- maintain performance and simplicity
