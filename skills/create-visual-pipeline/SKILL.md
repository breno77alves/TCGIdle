# Skill: create-visual-pipeline

## Purpose
Define a production-friendly visual pipeline for icons, portraits, card frames, and card layout components.

## Output Format
1. asset categories
2. directory structure
3. naming/export conventions
4. prompt templates where relevant
5. layout rules
6. consistency checklist

## Required Coverage
- creature portraits
- element icons
- tribe icons
- rarity badges
- card frame layers
- UI icons
- card layout zones

## Rules
- assets must work at browser scale
- readability beats detail density
- pipeline should support placeholder-to-final progression cleanly
